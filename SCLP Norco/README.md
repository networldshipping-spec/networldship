# Shell Chemical LP Norco — Website

This is a minimal static website scaffold for Shell Chemical LP Norco (mineral resources trading).

To preview locally:

1. Open `index.html` in a browser, or
2. Run a simple static server (Python 3):

```bash
python -m http.server 8000
# then open http://localhost:8000
```
